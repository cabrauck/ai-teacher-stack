# Obsidian Vault Workflow - Shaping Notes

Status: Planning

## Scope

Plan a local lesson folder workflow under `vault/Unterricht/` with Markdown planning notes, generated document references, curriculum links, and reflection notes.

## Decisions

- Use Obsidian-compatible Markdown as durable memory.
- Keep reflection examples anonymized and class-level.
- Link lesson plans to curriculum records where possible.
- Keep Claude-OS memory extraction as post-v1 backlog.

## Out of Scope

- Claude-OS memory bridge implementation.
- BYCS or OneDrive sync.
- Student-specific notes.
- Automatic document upload.

## Product Alignment

Supports the v1 goal of local files and Obsidian vault as primary state.
