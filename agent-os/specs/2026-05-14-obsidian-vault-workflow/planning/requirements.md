# Requirements: Obsidian Vault Workflow

Status: Planning

## Functional Requirements

- Create local lesson folder conventions under `vault/Unterricht/`.
- Support a lesson plan Markdown file.
- Support references to generated worksheet and solution DOCX files.
- Support a reflection Markdown file.
- Link lesson plans to curriculum records where possible.
- Add backlink-friendly metadata.

## Non-functional Requirements

- Keep files readable in Obsidian without custom tooling.
- Keep examples anonymized and class-level.
- Keep cloud sync out of the default workflow.

## Non-goals

- No Claude-OS memory bridge in v1.
- No BYCS or OneDrive sync.
- No student-specific records.
